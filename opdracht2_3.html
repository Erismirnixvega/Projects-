Welke HTML versie is gebruikt vanaf nu?

De laatste versie van HTML is HTML5 als een standaard versie.

Welke CSS versie is gebruikt vanaf nu?

Het recente versie van CSS is vanaf nu momenteel CSS3 als een belangrijkste versie.

Wat is het verschil tussen browsers in termen hoe ze websites laten zien? (Chrome, safari, firefox, etc.)

Wanneer een browser de data krijgt van HTML website inclusief CSS code en doet een parse proces maken aan twee boomstructuren: Document Object Model en een CSS Object Model.
Bij sommige browsers gebruiken ze een verschillende render-engines met eigen specfieke manier van processeren zoals Blink (Chrome en andere chromium browsers) en Gecko (Firefox).

Definitie van parseren: "Opsplitsen aan gehaalde datagegevens en maakt een datastructuur met een formaat zodat het programma gebruikt voor zijn doel."

Definitie van render-engine: "Software die webinformatie met toepassing van opmaakinformatie weergeeft op het beeldscherm."

Welke browser is popularist met hoeveelheid gebruik aan RAMs?

Het meeste gebruikte browser is Google Chrome, met ram gebruik van ongeveer 1000 MiB.

Maar het is mogelijk dat het kan lager of hoger gaat hangt af van hoe het werkt met uw computer.
