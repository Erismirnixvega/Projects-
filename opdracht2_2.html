Over HTML-elementen

Dit is een korte opstel over basis elementen van HTML.

Structuur van een HTML code

Elementen

H1 is de grootste kop dat wordt gebruikt meestal als een titel, p is ook gebruikt als een paragraph voor maken aan een alinea.

A is voor sturen naar soorten van links zoals: Telefoonnummers, hyperlinks, e-mails, secties, etc.

Img is gebruikt als fotos laten uitzien op uw site, vanuit een bron zoals lokale folder en vanuit een hyperlink bevatten met een bestandsformaat.

HTML kon ook een tabel hebben via een element namelijk: table (met rijen en kolommen voor zelf-instelling).

Er is ook een verschillende lijst elementen voor HTML. ul (Unordered list) en ol (Ordered list) zijn lijsten gebruikt voor makkelijke en kort manier om worden laten zien op uw site.
Het verschil tussen ul en ol is dat ul geeft een lijst niet in orde dat gebruikt een puntje maar ol gebruikt nummers als laten zien van woorden in een lijst gedaan in een ordelijke volgorde gebruikt zoals recepten of stappenplannen.
